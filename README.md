## 🌟 用《明朝那些事儿》的风格，讲述比特币的传奇历史

这是一个开源的中文比特币历史写作项目，致力于用生动有趣的叙事方式，为读者展现从 **1976年哈耶克的货币理论** 到 **2025年比特币生态繁荣** 的完整历史画卷。

### 我们将带您：
- 🕰️ **穿越时空** - 重访比特币诞生前后的关键历史时刻
- 👥 **走近人物** - 了解哈耶克、密码朋克、中本聪等关键人物
- 🔧 **理解技术** - 用通俗易懂的方式解释复杂的技术概念
- 💡 **感受理想** - 体会这场改变世界的货币革命背后的理想主义
