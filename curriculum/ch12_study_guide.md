# 延伸阅读：算力革命

> 对应章节：`正文/12_风起云涌：算力革命.md`

- [Laszlo 的 GPU 挖矿原始帖子（BitcoinTalk #124）](https://bitcointalk.org/index.php?topic=124.0) — Laszlo Hanyecz 分享用 OpenCL 在 GPU 上挖矿的代码和测试结果，开启了从 CPU 到 GPU 的算力迁移
- [Slush Pool 创建公告（BitcoinTalk #1497）](https://bitcointalk.org/index.php?topic=1497.0) — Marek Palatinus 在 2010 年 11 月发布的世界第一个比特币矿池的上线公告，解决了小矿工收入不稳定的"彩票问题"
- [比特币挖矿硬件演进史（Bitcoin Wiki）](https://en.bitcoin.it/wiki/Mining_hardware_comparison) — 从 CPU 到 GPU 到 FPGA 再到 ASIC 的完整硬件演进对比，展示算力军备竞赛如何在两年内彻底改变挖矿格局
