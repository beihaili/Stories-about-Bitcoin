# 延伸阅读：社区与工具

> 对应章节：`正文/05_初出茅庐：社区与工具.md`

- [BitcoinTalk forum](https://bitcointalk.org/) — Martti Malmi 于2009年11月创建的比特币论坛，比特币社区的第一个家，早期价格发现和技术讨论的核心场所
- [Martti Malmi (Wikipedia)](https://en.wikipedia.org/wiki/Martti_Malmi) — 芬兰开发者 Malmi 的生平，比特币第三号贡献者，搭建 bitcoin.org、编写 Linux 支持代码、挖出55000枚比特币的早期故事
- [Bitcoin v0.1 release announcement (Satoshi Nakamoto Institute)](https://satoshi.nakamotoinstitute.org/emails/cryptography/16/) — 中本聪2009年1月8日在密码学邮件组发布比特币 v0.1 版本的原始公告邮件
